Byte Kraft's Affiliate Bot

Personal Tool

Byte Kraft's Affiliate Bot is a personal automation tool built and operated by Byte Kraft for personal affiliate marketing on Pinterest. It automatically creates and posts Amazon affiliate product pins to Byte Kraft's own Pinterest boards.

What it does

  • Reads a curated list of Amazon product ASINs
  • Fetches product titles, prices, and images via the Amazon PA-API
  • Generates Pinterest-optimized pin images with product details
  • Writes SEO-optimized pin titles and descriptions (with mandatory FTC disclosure hashtags)
  • Posts the pins to Byte Kraft's personal Pinterest boards via the Pinterest API v5
  • Logs posted pins to prevent duplicates

Who operates this

This tool is a single-user personal project operated by Byte Kraft. It only accesses Byte Kraft's own Pinterest account and posts to its own boards. It does not collect, store, or process any third-party user data.

FTC Disclosure

All pins posted by this tool include #ad and #affiliate disclosures as required by US FTC guidelines.
