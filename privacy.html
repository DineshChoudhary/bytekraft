Privacy Policy

Last updated: March 14, 2026

This privacy policy applies to Byte Krafts Affiliate Bot, a personal automation tool operated by Byte Kraft at bytekrafts.github.io. This tool uses the Pinterest API to post affiliate product pins to Byte Krafts own Pinterest boards.

1. Scope

This is a single-user, personal-use tool. It is not a public application or service. It does not have end users, user accounts, or a sign-up flow. All API access is scoped exclusively to Byte Krafts own Pinterest account.

2. Data Collected

This tool does not collect personal data from any third party. The only data handled is:

  • Pinterest OAuth access token — stored locally in a .env file on Byte Krafts personal computer and used solely to authenticate API calls to Byte Krafts own Pinterest account.
  • Amazon product data — product titles, prices, and images fetched from the Amazon PA-API for the purpose of creating pin content. This data is not stored beyond the duration of a pipeline run.
  • Posted pin log — a local CSV file (logs/posted.csv) recording which ASINs have been posted, used only for deduplication. This file stays on Byte Krafts local machine.

3. How Data Is Used

All data described above is used solely to automate the creation and posting of Amazon affiliate product pins to Byte Krafts personal Pinterest boards. It is not used for analytics, advertising targeting, or any other purpose.

4. Data Retention

  • Pinterest access tokens are retained on Byte Krafts local machine until manually revoked or deleted.
  • Product data (titles, prices, images) is used transiently during a pipeline run and not persisted beyond that run.
  • Generated pin images are deleted immediately after being posted.
  • The posted-pin log CSV is retained indefinitely for deduplication purposes and is never shared with any third party.

5. Data Sharing

No personal data is shared with any third party. Pins are posted to Pinterest via the official Pinterest API under Byte Krafts own account. Affiliate links are generated via the Amazon Associates program and optionally routed through Geniuslink for geographic localization — neither service receives any end-user personal data through this tool.

6. Third-Party Services

This tool interacts with the following third-party APIs under their respective terms of service:

  • Pinterest — for posting pins
  • Amazon PA-API — for product data
  • Geniuslink (optional) — for affiliate link localization

7. Pinterest API Usage

This tool uses the Pinterest API solely to create pins on Byte Krafts own Pinterest boards. It does not read, store, or process data from any other Pinterest user's account.

8. Security

API credentials are stored in a local .env file that is excluded from version control via .gitignore. No credentials or personal data are transmitted beyond what is required for API authentication.

9. Changes to This Policy

If this policy changes materially, the updated date at the top of this page will be revised. Since this is a personal tool with no public users, no active notification process is required.
